Options
Format
Description There are 3 already supported formats :
Integer
This format defines a pre specified range from 0 to 359 with the step, equal to 1.
Options 'min', 'max' and 'step' are not taken into account;
Decimal
This format defines a pre specified range from 0 to 359 with the step, equal to 0.01.
Options 'min', 'max' and 'step' are not taken into account; Differs from 'integer' format only with the 'step' value;
Range
This format allows to specify any range with any step with the help of 'min', 'max' and 'step' options;
On plugin initialization, all the options are validated :
  • 'Min' and 'max' options validity is checked;
  • It's checked whether the range can be covered with the integer number of steps;
If there are any errors, then the invalid value is set to default.
Default value is 'integer';
Minimum
Description
Minimum
Specifies the minimum value of the slider;
This options is taken into account only for 'range' format;
Default value is '0';
Maximum
Description
Maximum
Specifies the maximum value of the slider;
This options is taken into account only for 'range' format;
Default value is '359';
Step
Description
Step
Determines the size or amount of each interval or step the slider takes between the min and max. The full specified value range of the slider (max - min) should be evenly divisible by the step.
This options is taken into account only for 'range' format;
Default value is '1' for 'integer' format and '0.01' for decimal;
Value
Description
Value
Specifies the initial value;
On plugin initialization, it's checked whether the value in in the range and can be targeted with the integer number of steps; If validation fail, it's reset to default.
This options is taken into account only for 'range' format;
Default value is '0';
Prefix
Description
Prefix
Specifies the prefix which will be shown in plugin before the current value;
Default value is '';
Suffix
Description
Suffix
Specifies the suffix which will be shown in plugin after the current value;
Default value is '';

Destroy
Description Removes the slider functionality completely. This will return the element back to its pre-init state.
Disable
Description Disables the slider.
Enable
Description Enables the slider.
Events
Start
Description Triggered when the user starts sliding.
Stop
Description Triggered when the user stops sliding.
Slide( value )
Description Triggered on every mouse move during slide. The value provided in the event as value represents the value that the handle will have as a result of the current movement.
Change( value )
Description Triggered if the value has changed or if the value is changed programmatically via the value method.
